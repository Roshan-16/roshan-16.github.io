Carry Lookahead Adder (CLA)

A Carry look ahead adder circuit using Manchester Carry Chain (MCC) was designed and implemented at the transistor-level using Verilog. The DC and transient analysis, that provide the switching delay and other transistor parameters, was done using Ngspice. The parasitic capacitances due to the transistors layout were found out using Magic VLSI layout tool.
Software/Tools used: Icarus Verilog, Ngspice, Magic VLSI layout tool
